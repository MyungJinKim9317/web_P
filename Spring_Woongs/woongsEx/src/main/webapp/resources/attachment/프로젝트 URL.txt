쿠폰 페이지 url
coupon - 쿠폰페이지 이동
couponIssue - 쿠폰 발행 url 
couponView - 쿠폰 보여주는 Ajax 
couponEnroll - 유저가 쿠폰을 등록하는 url
couponDelete - 관리자가 발행한 쿠폰을 삭제하는 메소드

포스트 관련 url
/upload - 게시물 업로드 페이지 이동
/tag - 대분류 불러오는 ajax url
/subtag - 대분류 넘버로 소분류 불러오는 ajax url
/insertPost - 이미지랑 값 받아서 게시물 등록하는 url
/post GET - 해당 게시물 번호로 게시물 정보를 받아오는url
/post POST - 모든 게시물을 가져오는 url
/category GET - 해당 카테고리에 해당하는 게시물을 가져오는 url
/subcategory GET - 해당 대분류, 소분류에 해당하는 게시물을 가져오는 url

수익관리 관련 url
/income - 수익관리 페이지로 이동
/findIncomeByUserId - userId별로 수익관리 볼수 있는 url
/findAccountInfoByUserId - userId 별로 계좌정보를 볼수 있는 url
